# 📚 Book Recommendation System

## Overview

A Machine Learning-based Book Recommendation System that recommends books based on user interests using multiple recommendation techniques. The project uses the **Book-Crossing Dataset** and implements both traditional and hybrid recommendation approaches.

## Features

* Popularity-Based Recommendation
* Average Weighted Rating Recommendation
* User-Item Collaborative Filtering
* Correlation-Based Recommendation
* K-Nearest Neighbors (KNN) Recommendation
* Content-Based Recommendation (TF-IDF)
* Hybrid Recommendation (Collaborative + Content)

## Dataset

The project uses the **Book-Crossing Dataset**, which contains:

* Books
* Users
* Ratings

The data is cleaned and preprocessed before training the recommendation models.
